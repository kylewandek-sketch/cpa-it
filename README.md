# Chromebook QR Scanner

A self-contained web page that uses your device camera to scan Chromebook QR/serial
codes, de-duplicates them, and exports a CSV formatted for the classroom cart sheets.

- No install, no dependencies (jsQR embedded), works offline.
- Open on a phone or laptop, tap **Start camera**, hold each QR in the green box.
- **Export format** dropdown → **Download CSV** (or **Copy all**).

Live: served via GitHub Pages (see the repo's Pages URL).
